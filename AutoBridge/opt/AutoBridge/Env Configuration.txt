
GM61 AutoBridge .env configuration instructions

Developed by GM61 Limited

www.gm61.co.uk


This document provides the user with a set of instructions on how to correctly configure the .env file for GM61 AutoBridge


If you cannot see the .env file you will need to select "Show Hidden Items" in your file explorer.


Here is an example of the .env file:

WATCH_FOLDER=/Users/nick/Desktop/DemoWasher
START_DATE=2025-01-01
REMOVE_SPACES=true
REMOVE_HASHTAGS=true
MAX_RETRIES=2
API_URL=https://sdl.ihss.co.uk/api/v1/cycles/external/upload
MACHINE_CODE=GETINGE
SUBFOLDER=false
XML_TAG=MACHNAME
POLL_INTERVAL=10
MOVE_FILE=true
MOVE_FILE_PATH=/Users/nick/Desktop/DemoWasher/processed

# Auth credentials for token retrieval
AUTH_URL=https://sdl.ihss.co.uk/api/six-auth/users/login
AUTH_USERNAME=???
AUTH_PASSWORD=???


1. The watch folder is the file path of the filder that contains the files to be uploaded

2. The start date is used to reference the files that should be uploaded, only files created after the start date will be uploaded with AutoBridge

3. If Remove spaces is set to true, then if there are any spaces in file paths these will be removed.

4. If remove hastags is set to true, this will remove any hastags in file paths.

5. Max retries states how many times AutoBridge will attempt to upload a file before marking it as unsuccessful

6. API url is the endpoint that files are sent to

7. Machine code is the code for the washer, this is usually the brand.

8. If subfolder is true, this will look for any subfolders within the watch folder. This will then use the subfolder name as the machine serial during upload.

9. xml tag is only required if subfolder is false, if there is no subfolder then AutoBridge will look in the xml file to find the machine serial, this variable is used to define that field within the xml file.

10. Poll interval is used to configure the delay for AutoBridge to search for new files in seconds.

11. If move file is true, once a file has been uploaded it will be moved from the origional folder.

12. move file path is only required if move file is true, this defines the path that files will be moved to once processed.

13. Auth URL is the endpoint to obtain Authentication to the web service.

14. Username for the endpoint

15. Password for the endpoint
